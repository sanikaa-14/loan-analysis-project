**Overview:**
This project analyzes a loan dataset to identify key factors influencing loan approval decisions using SQL on Databricks
**Objective:**
To understand how income, credit score, and education level affect loan approval.
**Dataset Features:**
1)Age
2)Gender
3)Income
4)Credit Score
5)Education Level    
6)Loan Status
**Key Insights:**
1)Higher income applicants are more likely to get approved
2)Credit score is the strongest factor influencing approval
3)Higher education levels show better approval rates
**Conclusion:**
Loan approval decisions are primarily influenced by financial stability indicators like income and credit score.
